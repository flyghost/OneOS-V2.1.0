INTRODUCTION

OneNET-MQTTS component is a complete application of the OneOS operating
system for developers.

OneNET-MQTTS component has the following features:
1. Automatic device registration;
2. Reliable connection of device (heartbeat packet, network reconnection\ 
   and device reconnection);
3. The message supports TLS transmission;
4. Subscription;
5. Publish - upload data (specific JSON format). Data points support int,\
   float, string and object;
6. Publish image information getting;
7. Publish - image information update;
8. Platform command response.
